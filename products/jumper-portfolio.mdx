## What is Jumper Portfolio?

Jumper Portfolio is a cross-chain portfolio aggregation layer integrated directly into Jumper.

It provides users with a consolidated view of their tokens and DeFi positions across supported blockchains and protocols.

<Note>
  Jumper Portfolio helps users monitor their assets and DeFi positions across chains in one dashboard. Portfolio does not custody user funds.
</Note>

## What problem does Jumper Portfolio solve?

DeFi users typically hold:

- Tokens on multiple chains
- Positions across multiple protocols
- Assets in different wallets

Tracking these manually requires switching between explorers, dashboards, and applications.

Jumper Portfolio consolidates this information into a single interface.

---

## What can you check on Jumper Portfolio?

Jumper Portfolio helps you keep trakc of:

- Total portfolio value
- Asset balances across chains
- Protocol attribution for each position
- Position type (lending, LP, staking, vault, etc.)
- Asset allocation distribution
- Portfolio performance over time
- Transaction history

## Relationship to other Jumper products

Jumper Portfolio tracks positions created through:

- Same-chain swaps
- Cross-chain swaps
- Jumper Earn deployments

It provides the visibility layer for Jumper's full capital journey.
